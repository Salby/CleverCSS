Navigation Bar

CleverCSS provides a navigation bar that is both easy to use and versatile

Basic navigation bar

<nav class="navBar">
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Contact</a></li>
  </ul>
</nav>

Navigation bar with "burger menu" for mobile devices

The mobile menu button will hide itself when the navigation bar isn't collapsed

<nav class="navBar">
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
    <li><a href="#">Contact</a></li>
    <li id="drawerButtonItem"><button id="drawerButton">...</button></li>
  </ul>
</nav>

Buttons

Flat Button

A button that sits flat on it's parent. Good for dialogs and small actions, like cancelling an action.

<button class="buttonFlat">BUTTON</button>

Raised Button

A button that is raised from it's parent. Good if you have a few options that need to be more visible than the flat button.

<button class="buttonRaised">BUTTON</button>

Action Button

This button is also raised from it's parent. It's good for highlighting important actions. It's a good idea not to use this button too often, as it would start to seem insignificant compared to the other buttons. The action button needs an icon that represents the action.

<button class="buttonAction">BUTTON</button>

Dropdown

The dropdown can be used to place a bunch of actions in a small space. It's good if you want to save space for something like a mobile device

<div class="dropdown">
 <button class="buttonDropdown"></button>
 <div class="dropdownContent"></div>
  <ul class="dropdownMenu">
   <li><a>Option 1</a></li>
   <li><a>Option 2</a></li>
   <li><a>Option 3</a></li>
  </ul>
 </div>
</div>

Hover over the button to trigger the dropdown.

Icons

You can use any icon pack you like with CleverCSS, although Google's material icons and Font Awesome are recommended

Dependencies

JQuery is needed for certain elements to work correctly.
Jquery can be installed from the official JQuery website

You can also use a link from Google APIS to place in your <head>

<script src='https://ajax.googleapis.com/ajax/libs/jquery/3.1.1/jquery.min.js'></script>
