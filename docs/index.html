Navigation

Short navigation bar

<nav class="navbar navbar-dark navbar-short">
  <ul class="navbar-nav navbar-left">
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
  </ul>
</nav>

Horizontal alignment

You can also use the class navbar-right to move the links to the right side of the navigation bar

<nav class="navbar navbar-dark navbar-short">
  <ul class="navbar-nav navbar-right">
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
  </ul>
</nav>

Brand

By default, links in the navbar disappear once the screen size gets too small. To keep a link from disappearing use the class brand-item

<nav class="navbar navbar-dark navbar-short">
  <ul class="navbar-nav navbar-left">
    <li class="brand-item"><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
  </ul>
</nav>

Tall navigation bar

A tall navigation bar is more spatious, but takes up a lot of space on smaller devices

<nav class="navbar navbar-dark navbar-tall navbar-tall">
  <ul class="navbar-nav navbar-left">
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
  </ul>
</nav>

You can use navbar-right and brand-item just like on the short navigation bar

Drawer

The drawer is a menu that hides links until they are revealed by the user

The first thing you'll need to use the drawer is a button that can reveal it. You can put this button in your navigation bar like this:

<nav class="navbar navbar-dark navbar-short">
  <button id="drawer-open" class="drawer-open-left"><i>...</i></button>
  <ul class="navbar-nav navbar-left">
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
  </ul>
</nav>

You can add the class mobile-only to #drawer-open if you only want it to be visible on touch screens

You can use drawer-open-right on #drawer-open if you want to move it to the right side of your navigation bar

Next you'll need to add the drawer to your markup like this:

<nav class="navbar navbar-dark navbar-short">
  <button id="drawer-open" class="drawer-open-left"><i>...</i></button>
  <ul class="navbar-nav navbar-left">
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
    <li><a href="#">Link</a></li>
  </ul>
</nav>
<!--DRAWER-->
<div id="mobile-drawer" class="drawer drawer-left">
  <div class="drawer-content drawer-dark">
    <div class="drawer-header drawer-dark">
      <button id="drawer-close">...</button>
    </div>
    <ul class="drawer-nav drawer-nav-left">
      <li><i>...</i> <a href="#">Link</a></li>
    </ul>
  </div>
</div>

Remember that you can use drawer-light instead of drawer-dark and so on. You can also replace left with right in classes that use horizontal alignment

Buttons

Flat Button

A button that sits flat on it's parent. Good for dialogs and small actions, like cancelling an action.

<button class="button-flat">button</button>

Raised Button

A button that is raised from it's parent. Good if you have a few options that need to be more visible than the flat button.

<button class="button-raised">button</button>

Action Button

This button is also raised from it's parent. It's good for highlighting important actions. It's a good idea not to use this button too often, as it would start to seem insignificant compared to the other buttons. The action button needs an icon that represents the action.

<button class="button-action">...</button>

Modals

Modals are small windows that can contain information not neccesary to be presented immedeately

The modal button

The first thing you'll need is a button that can activate the modal

<button class="button-flat buttonModalOpen" data-target="#modal1">Open Modal</button>

The data-target attribute is used to point to the modal that you want to open

The modal window

The modal itself needs an id attribute. This is how you are able to target different modals with different buttons

<div class="modal" id="modal1">
  <div class="modalContentLight">
    <div class="modalHeaderLight">
      <h2>Header</h2>
      <p>Comment</p>
    </div>
    <div class="modalFooterLight">
      <button class="button-flat buttonModalClose" data-target="#modal1">Close Modal</button>
    </div>
  </div>
</div>

Forms

<form class="form-basic">
  <!--TEXTFIELD-->
  <div class="form-group">
    <input type="text" id="text" required>
    <label class="control-label" for="text">Textfield</label><i class="bar"></i>
  </div>
  <!--SELECTION-->
  <div class="form-group">
    <select id="select">
      <option value="1">Option 1</option>
      <option value="2">Option 2</option>
    </select>
    <label class="control-label" for="select">Selection</label><i class="bar"></i>
  </div>
  <!--RADIO BUTTONS-->
  <div class="form-radio">
    <!--RADIO BUTTON CHECKED-->
    <div class="radio">
      <label>
        <input checked type="radio"><i class="helper">Radio button
      </label>
    </div>
    <!--RADIO BUTTON UNCHECKED-->
    <div class="radio">
      <label>
        <input type="radio"><i class="helper">Radio button
      </label>
    </div>
  </div>
  <!--CHECKBOX CHECKED-->
  <div class="checkbox">
    <label>
      <input checked type="checkbox" name="checkbox"><i class="helper"></i>Checkbox
    </label>
  </div>
  <!--CHECKBOX UNCHECKED-->
  <div class="checkbox">
    <label>
      <input type="checkbox" name="checkbox"><i class="helper"></i>Checkbox
    </label>
  </div>
  <!--TEXTAREA-->
  <div class="form-group">
    <textarea name="textArea" rows="4" cols="40"></textarea>
    <label class="control-label" for="textArea">Textarea</label><i class="bar"></i>
  </div>
</form>
